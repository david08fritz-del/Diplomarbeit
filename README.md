# Versichert und trotzdem nicht abgesichert

Praktischer Teil der HAK-Diplomarbeit (BHAK Bludenz) zum Thema **Versorgungslücken
zwischen gesetzlicher Sozialversicherung und privater Vorsorge** — für junge Leute
(18–25) in Österreich.

## Was das ist

Eine hochwertige **Story-Webseite** rund um ein eigenständiges **Browser-Lernspiel
„Verteidige dein Leben"**. Die Webseite ist die Bühne (erklärt, zeigt Videos,
präsentiert die Umfrage-Ergebnisse); das Spiel ist die Attraktion.

**Kernbotschaft:**
1. Versichert ≠ abgesichert — man glaubt sich gedeckt, wo man es nicht ist.
2. Die *richtige* Absicherung, nicht die *meiste* — sie hängt an der Lebenssituation.
